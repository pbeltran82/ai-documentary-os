# AI Documentary OS

An AI-first production system for creating cinematic, faceless YouTube documentaries and repurposing them into short-form content.

## Mission

Build a scalable documentary studio that uses AI to improve storytelling quality, viewer retention, production speed, and automation.

## Current Priority

Produce Episode 1 while documenting a repeatable workflow that can later become a largely autonomous content-production pipeline.

## Core Goals

1. Create premium long-form YouTube documentaries.
2. Repurpose each episode into Shorts, TikToks, Reels, and social posts.
3. Automate research, scripting, voiceover, visuals, editing, publishing, and performance analysis.
4. Build reusable prompts, templates, assets, and production systems.
5. Optimize every decision for retention, storytelling, scalability, and long-term channel growth.

## Planned Production Pipeline

```text
Idea Discovery
    ↓
Research and Source Verification
    ↓
Story Architecture
    ↓

    ↓
Voiceover Generation
    ↓
Visual Asset Production
    ↓
Editing and Sound Design
    ↓
Quality Control
    ↓
Publishing and Distribution
    ↓
Analytics and Continuous Improvement
```

## Repository Structure

```text
docs/               Project strategy and operating procedures
episodes/           Episode research, scripts, scenes, and assets
prompts/            Reusable AI prompts
src/                Future automation services and application code
templates/          Reusable production templates
```

## Guiding Principles

- Viewer retention comes first.
- Storytelling quality must remain cinematic and credible.
- Verify factual claims and preserve source traceability.
- Automate repetitive work without lowering quality.
- Build reusable systems instead of one-off processes.
- Avoid features that do not improve quality, growth, or autonomy.
